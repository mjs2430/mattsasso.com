# mattsasso.com — Project Context

Personal portfolio website for Matt Sasso. Static front-end, no build step.

- **Stack:** Vanilla HTML/CSS/JavaScript, jQuery, custom web components (`main-header`, `main-footer`, `work-card`)
- **Architecture:** Client-side filtering on the project gallery; project detail pages under `/projects/`; Google Analytics integration
- **Hosting:** Static hosting (no server-side logic)

No active ongoing work as of 2026-06-15.
