podcast creator tool and templates

2017 | ui, ux, html, js, css visit the site

I collaborated with a coworker to create a tool that allowed our journalists to use a WYSIWYG editor to create multiple new podcast cards for our markets. The tool would then output the needed code, and using the directions in my documentation, the user could copy and paste that code into the proper place in McClatchy's CMS. I also created the templates needed for each market.

I wrote extensive documentation, step-by-step guides with video, and wrote most of the logic in the code. I designed and coded each template as well, seen in this example page.
