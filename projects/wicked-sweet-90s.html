wicked sweet 90s

2018 | ui, ux, html, js, css, wordpress visit the site

Balancing the needs of the client (90s styling and feel), with the needs of a user (clean, modern experience) was a fun challenge on this project. This is a site for a repeat client who already had liketotally80s.com, a successful era-blog. He wanted to build sites based on the 1970s and the 1990s. Using Wordpress, I customized a theme and created a color palette and logo that represented the 90s in all its... unique glory.

As with almost all of my Wordpess sites, I also acted as admin. I setup user roles, ran demos, taught users how to use the appropriate features, maintained plugins, ran database backups, etc.
