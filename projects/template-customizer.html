template customizer

2017 | ui, ux, html, js, css visit the site

I created a tool for McClatchy markets to customize settings and styles for their niche website templates, save them, and share them with our design team. This optimized the workflow between our design team and our markets' template requirements.

This tool was built with the Materialize css framework for the layout and JavaScript and JQuery for the logic. Users could pick from styling options and the tool would save everything to a custom URL that they could share with our team and we would see all the options they picked.
